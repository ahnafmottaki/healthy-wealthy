## 💻 How to Run Locally

### 1. Clone the Repo

```bash
git clone https://github.com/FreeCodeJunction/healthy-wealthy.git
cd healthy-wealthy
```

### 2. Install Dependencies

```bash
cd healthy-wealthy
npm install
```



### 4. Start Servers

```bash
npm run dev
```
